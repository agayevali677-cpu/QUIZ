Addım-addım necə edəcəksən:

1. Input və düymə yarat

HTML-də istifadəçi üçün bir yazı sahəsi (input) və göndərmə düyməsi (button) lazım olacaq.

html
<input type="text" id="istifadeciYazi" placeholder="Buraya yazı yazın">
<button onclick="sekliGoster()">Göndər</button>
<div id="cavab"></div>

2. JavaScript funksiyası yaz

  • Funksiya düyməyə klik ediləndə işləyəcək.

  • İstifadəçinin yazısını götürəcək.

  • Sonra div-in içinə həm yazını, həm də bir şəkli əlavə edəcək.

html
<script>
function sekliGoster() {
    let yazi = document.getElementById("istifadeciYazi").value;
    let cavabDiv = document.getElementById("cavab");

    // Burada sənin şəkil faylının adını yazmalısan
    let sekil = "menimsekil.jpg";

    cavabDiv.innerHTML = "<p>Sən yazdın: " + yazi + "</p>" +
                         "<img src='" + sekil + "' width='300'>";
}
</script>

3. Şəkil faylını qoy

  • Eyni qovluqda (sənin HTML faylının yanında) menimsekil.jpg adlı şəkil faylı saxla.

  • Əgər başqa yerdədirsə, yolunu yaz:

    • Məs: img/menimsekil.jpg

    • və ya tam yol: C:/Users/.../sekil.jpg

🔑 Qaydası budur:

  • input → yazını götürürsən

  • innerHTML → cavab yerinə yazı + şəkil qoyursan

  • img src → şəkilin yolunu göstərir

Variant 2: Ayrı-ayrı fayllar

Əgər sən CSS və JavaScript kodunu ayrıca faylda saxlamaq istəyirsənsə, onda 3 faylın olacaq:

  1. index.html

    html
    <!DOCTYPE html>
<html lang="az">
<head>
    <meta charset="UTF-8">
    <title>Nümunə</title>
    <link rel="stylesheet" href="style.css"> <!-- CSS faylı qoşuldu -->
</head>
<body>
    <input type="text" id="istifadeciYazi" placeholder="Buraya yazı yazın">
    <button onclick="sekliGoster()">Göndər</button>
    <div id="cavab"></div>

    <script src="script.js"></script> <!-- JS faylı qoşuldu -->
</body>
</html>

  2. style.css

    css
    body {
    font-family: Arial, sans-serif;
}
#cavab {
    margin-top: 20px;
    text-align: left;
}

  3. script.js

    javascript
    function sekliGoster() {
    let yazi = document.getElementById("istifadeciYazi").value;
    let cavabDiv = document.getElementById("cavab");

    let sekil = "menimsekil.jpg"; // şəkil eyni qovluqda olmalıdır

    cavabDiv.innerHTML = "<p>" + yazi + "</p>" +
                         "<img src='" + sekil + "' width='300'>";
}

Vacib məqamlar ⚡

  • Üç fayl (HTML, CSS, JS) eyni qovluqda olmalıdır.

  • Şəkil (menimsekil.jpg) də həmin qovluğa qoy ki, rahat işləsin.

  • index.html-i açanda hər şey işləyəcək.

İndi sən istəsən hamısını bir yerdə saxla (asan yol), istəsən də ayrı-ayrı fayl kimi yaz (professonal yol).
